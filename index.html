Urban Garden - Automatic Plant Watering System

This project is an automatic plant watering system that uses a moisture sensor to automatically sense the moisture content in the soil and waters it using a water pump if the moisture content is less than the limit which is predefined. The system is connected to an application on our mobile device which sends a notification to the owner about the plant whenever it has been watered. From the application, we can also choose the type of plant and keep track of our plant to keep it healthy and grow.

Java Based Chatbot Application for Hotel Services

This project is a chatbot which is a software application used to conduct an on-line chat conversation via text for a hotel booking service. This system not only makes booking hotels easy but it also provides an on-line chat conversation via text by adding various features like asking the user for his/her meals, to choose between AC and NON-AC rooms and so on... In the pandemic moreover, providing a contactless contact is achieved due to the use of chatbots.
